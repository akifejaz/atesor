# Atesor AI (RISC-V Porting Multi-Agent System)

> Atesor AI is a LangGraph-based, multi-agent orchestration system that ports upstream software projects to RISC-V inside a Dockerized Alpine/riscv64 sandbox, combining deterministic scripted analysis with LLM-guided planning, build execution, fixing, and escalation.

Atesor has two execution layers: deterministic scripted ops (clone/update, detection, dependency extraction, environment probing) and agentic orchestration (planner/supervisor/scout/builder/fixer/summarizer). For implementation work, state (`AgentState`) is the system contract and routing decisions in the graph should stay consistent with error severity and escalation behavior.
